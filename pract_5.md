# Задача 1
Доказательство:
N = n1​n2…nk​
Ni = N / ni для i = 1, 2, …, k
Так как ni попарно взаимно просты, то:
gcd(ni, Ni) = 1 для всех i
Существуют целые числа ti такие, что tiNi ≡ 1(mod ni) (поскольку Ni обратимо по модулю ni)
