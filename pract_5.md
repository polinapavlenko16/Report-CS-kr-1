264 * t2 ≡ 1 (mod 13)  
264 ≡ 4 (mod 13)  
4 * t2 ≡ 1 (mod 13) ⇒ t2 = 10 (так как 4 * 10 = 40 ≡ 1 (mod 13))  
143 * t3 ≡ 1 (mod 24)  
143 ≡ 23 (mod 24)

x0 = 5 * 312 * 3 + 4 * 264 * 10 + 3 * 143 * 23 = 5 * 936 + 4 * 2640 + 3 * 3289 = 4680 + 10560 + 9867 = 25107

x ≡ 25107 (mod 3432)

# Задача 4  
16-битовое число K=39027, схема с 4 участниками.  
Генератор директор получает случайное 16-битное S1  
Заместитель директора получает случайное 16-битное S2  
Главный бухгалтер получает случайное 16-битное S3  
Заместитель главного бухгалтера получает S4 = K − S1 − S2 − S3 (mod 2 ^ 16)  
Выберем случайные значения (например):  
S1 = 15000  
S2 = 23000  
S3 = 5000  
S4 = 39027 − 15000 − 23000 − 5000 (mod 65536) = 39027 − 43000 = −3973 (mod 65536)  
−3973 ≡ 65536 − 3973 = 61563 (mod 65536)  
Проверка восстановления:  
S1 + S2 + S3 + S4 = 15000 + 23000 + 5000 + 61563 = 104563  
104563 (mod 65536) = 104563 − 65536 = 39027  
Ответ:  
S1 = 15000  
S2 = 23000  
S3 = 5000  
S4 = 61563
