# Задание 2  
ARP используется для преобразования IP-адресов в MAC-адреса в локальной сети. Можно извлечь информацию о MAC-адресах устройств, IP-адресах, признаках ARP-спуфинга.  
TCP используется для обеспечения надежной, упорядоченной и проверенной доставки данных. Можно извлечь информацию о номерах портов, флагах (SYN, ACK, FIN), последовательности, полезной нагрузке.  
UDP используется для обеспечения быстрой, но ненадежной доставки данных без установки соединения. Можно извлечь информацию о номерах портов, длине данных, контрольной сумме, полезной нагрузке.  
HTTPS - это защищенная версия HTTP, использует шифрование TLS/SSL. Используется для безопасной передачи данных в интернете. Можно извлечь информацию о зашифрованных данных, доменном имени (SNI), сертификатах.  
TLS используется для обеспечения шифрования и аутентификации для защищенных соединений. Можно извлечь информацию о версии TLS, cipher suites, сертификатах, ключах шифрования.
