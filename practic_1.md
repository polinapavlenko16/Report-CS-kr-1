## Вопросы после задания 1

1. Моноалфавитным. Каждый символ исходного текста заменяется на один и тот же символ шифрованного текста согласно фиксированному сдвигу.
2. Симметричной. Для шифрования и дешифрования используется один и тот же ключ (значение сдвига N).
3. Для русского алфавита (33 буквы): N ∈ {1, 2, ..., 32}
Для английского алфавита (26 букв): N ∈ {1, 2, ..., 25}
Ключ N = 0 и N = 33 (для русского) не имеют смысла, так как не производят шифрования.
4. Достоинства:
· Простота реализации и понимания
· Быстрое выполнение операций
· Малое пространство ключей (легко запомнить)
Недостатки:
· Очень слабая криптостойкость
· Легко взламывается методом полного перебора (brute force)
· Уязвим к частотному анализу
· Не обеспечивает конфиденциальности в современных условиях
5. Математическое описание шифра Цезаря:
Шифрование:
Сопоставим буквы алфавита с числами от 0 до M−1 (M = 33).
Шифрование: E(P) = (P + N) mod M
Дешифрование: D(C) = (C − N) mod M
где P — номер буквы открытого текста, C — номер буквы шифртекста, N — ключ.

## Вопросы после задания 2

1. Полиалфавитный. Одна и та же буква открытого текста может шифроваться разными буквами в зависимости от позиции.
2. Симметричный. Для шифрования и дешифрования используется один и тот же ключ.
3. Пространство ключей шифра Виженера:
· Длина ключа: 1 до L символов (L ≈ длина текста)
· Каждый символ ключа: 33 возможных значения (для русского алфавита)
· Общее число ключей: 33 + 33² + 33³ + ... + 33^L
4. Достоинства:
· Устойчив к частотному анализу (по сравнению с моноалфавитными шифрами)
· Прост в ручной реализации
· Криптостойкость растёт с длиной ключа
Недостатки:
· Взламывается методом Казиски (при коротком повторяющемся ключе)
· Требует безопасной передачи ключа
· При коротком ключе уязвим к статистическому анализу
5. Шифр Виженера — это последовательность шифров Цезаря с разными значениями сдвига, задаваемыми ключевым словом.
6. Математическое описание:
Шифрование:
C_i = (P_i + K_{i mod m}) mod N
Дешифрование:
P_i = (C_i - K_{i mod m}) mod N
где:
· P_i — номер буквы открытого текста
· C_i — номер буквы шифртекста
· K_j — номер j-й буквы ключа
· m — длина ключа
· N — мощность алфавита (33)
· i mod m — позиция в ключе для i-й буквы текста
